┣━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┫
┃                                                                                          ┃
┃  POST /api/sync (Vercel Serverless Function)                                            ┃
┃  ─────────────────────────────────────────────────────────────────────────────────      ┃
┃                                                                                          ┃
┃  For each request (chunk):                                                              ┃
┃                                                                                          ┃
┃  1. Receive chunk (4 MB base64)                                                         ┃
┃  2. Validate headers (X-API-KEY)                                                        ┃
┃  3. Decode base64 → binary (4 MB)                                                       ┃
┃  4. Store in temporary storage:                                                         ┃
┃     ├─ In-memory (simple): sessions[sessionId][chunkNumber] = buffer                    ┃
┃     └─ Redis (resilient): await redis.set(sessionId:chunk:1, buffer, EX 86400)         ┃
┃  5. Check if all chunks received (totalChunks matches)                                  ┃
┃  6. If complete:                                                                        ┃
┃     ├─ Concatenate chunks: Buffer.concat([chunk1, chunk2, ...chunk13])                  ┃
┃     ├─ Decompress: pako.ungzip(concatenated) → 210 MB JSON                            ┃
┃     ├─ Parse JSON: JSON.parse(decompressed) → { products, stocks, ... }                ┃
┃     ├─ Validate schema (optional)                                                      ┃
┃     ├─ Store in Supabase                                                                ┃
┃     └─ Return success response                                                          ┃
┃                                                                                          ┃
┃  Function specs:                                                                        ┃
┃  • Runtime: Node.js 18+                                                                 ┃
┃  • Memory: 3 GB available (using ~100 MB)                                               ┃
┃  • Timeout: 30 seconds (handling 4 MB in ~2-3s)                                        ┃
┃  • Cost: $0.000002 per invocation (basically free)                                      ┃
┃                                                                                          ┃
┃  ╔═══════════════════════════════════════════════════════════════════════════════════╗  ┃
┃  ║  Example Handler (TypeScript/Node.js)                                           ║  ┃
┃  ║  ───────────────────────────────────────────────────────────────────────────── ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║  export default async function handler(req, res) {                            ║  ┃
┃  ║    if (req.method !== 'POST') return res.status(405).end();                   ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║    const { sessionId, chunkNumber, totalChunks, data, siteId } = req.body;   ║  ┃
┃  ║    const sessions = {}; // or Redis.get()                                     ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║    // Decode base64                                                            ║  ┃
┃  ║    const buffer = Buffer.from(data, 'base64');                                ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║    // Store chunk                                                              ║  ┃
┃  ║    sessions[sessionId] = sessions[sessionId] || {};                            ║  ┃
┃  ║    sessions[sessionId][chunkNumber] = buffer;                                 ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║    // Check if complete                                                        ║  ┃
┃  ║    if (Object.keys(sessions[sessionId]).length === totalChunks) {              ║  ┃
┃  ║      // Assemble                                                                ║  ┃
┃  ║      const chunks = Array.from({ length: totalChunks }, (_, i) =>             ║  ┃
┃  ║        sessions[sessionId][i + 1]                                             ║  ┃
┃  ║      );                                                                         ║  ┃
┃  ║      const assembled = Buffer.concat(chunks); // 45 MB                        ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║      // Decompress                                                              ║  ┃
┃  ║      const decompressed = pako.ungzip(assembled); // 210 MB                   ║  ┃
┃  ║      const json = JSON.parse(decompressed.toString()); // Parse               ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║      // Store in Supabase                                                      ║  ┃
┃  ║      await supabase                                                            ║  ┃
┃  ║        .from('sync_backups')                                                   ║  ┃
┃  ║        .insert({                                                               ║  ┃
┃  ║          session_id: sessionId,                                               ║  ┃
┃  ║          site_id: siteId,                                                     ║  ┃
┃  ║          compressed_data: assembled, // Store compressed                      ║  ┃
┃  ║          compressed_size: assembled.length,                                   ║  ┃
┃  ║          original_size: decompressed.length,                                  ║  ┃
┃  ║          timestamp: new Date(),                                               ║  ┃
┃  ║        });                                                                     ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║      delete sessions[sessionId]; // Clean up                                  ║  ┃
┃  ║      return res.status(200).json({ success: true });                          ║  ┃
┃  ║    }                                                                            ║  ┃
┃  ║                                                                                 ║  ┃
┃  ║    return res.status(202).json({ accepted: true }); // Still waiting          ║  ┃
┃  ║  }                                                                              ║  ┃

┃  ╚═══════════════════════════════════════════════════════════════════════════════════╝  ┃
